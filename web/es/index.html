Página oficial de MyQtt

MyQtt es un stack Open Source profesional del protocolo MQTT, escrito en ANSI C, que soporta las versiones v3.1.1 y v3.1, especializado en el soporte para crear servidores MQTT. MyQtt tiene un diseño modular que permite crear servidores MQTT usando el API proporcionada por libMyQtt (la cual está compuesta por varias librerías: libmyqtt, libmyqtt-tls, libmyqtt-websocket). También se proporciona un servidor MQTT listo para usar llamado MyQttD que está construido usando libMyQtt. El servidor MyQttD es extensible añadiendo nuevos plugins escritos en C.

Existe una interfaz Python para libMyQtt llamada PyMyQtt de manera que es posible escribir servidores MQTT completos en Python.

MyQtt está centrado en la seguridad, está muy comprobado y es estable en cada revisión (tests de regresión concienzudos son usados para comprobar todos los componentes libMyQtt, MyQttD and PyMyQtt) para asegurar que las características y el funcionamiento proporcionado es estable y robusto (sin sorpresas y con funcionalidad predecible después de cada actualización).

MyQtt ha sido creado por ASPL (Advanced Software Production Line, S.L.), y está licenciado bajo los términos de la LGPL 2.1 (de manera que se puede construir aplicaciones OpenSource y comerciales).

Acerca de Aspl

Aspl (Advanced Software Production Line) es una empresa española que proporciona soporte y servicios para introducir productos open source dentro de las organizaciones, especialmente GNU/Linux.

Fundada en el 2000, su principal línea de desarrollo es proporcionar soporte para el desarrollo de software así como soporte para instalar servidores GNU/Linux que proporcionan distintos servicios.

Visite nuestra página en http://www.aspl.es para saber más sobre nosotros.
